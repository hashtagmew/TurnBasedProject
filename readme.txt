The scene "map-editor" is semi-functional; the scene can be modifed with the tab TileEditor which can be found in the Windows menu. You can paint and erase simple tiles.

On Windows, WASD moves the camera, mouse wheel zooms, left click selects and right click deselects.
On Android, swiping moves the camera, pinching zooms, touching selects and quad-touching deselects (temporary).
